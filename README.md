Name of the product: Todo App

Introduction: This app is an android based app used for daily tasks. This App lets a user create, read, update and delete various tasks. A task has a task description, date of task can be in incomplete and completed. 



Requirement:

•	Android Studio 3.6 or newer

•	Android 4.4+


Features:

•	Add new todo

•	Pick a date from calendar

•	Mark todo as completed / uncompleted

•	Edit todo

•	View todo

•	Delete todo



Framework used:

•	MVVM (Model View ViewModel)


Database:

•	SQLite
